## 一、品牌核心表达

### 1.1 一句话定位

**中文**：UniMind.ai 是 Agent 驱动的新一代智能教育基础设施——从教师出题、学生刷题、评分批改到知识追踪，全链路 Agent 化。对话即学习，Agent 即基建。

**英文**：UniMind.ai is an Agent-driven next-generation education infrastructure — full-chain Agent automation from question generation and grading to knowledge tracking. Dialogue is learning, agents are infrastructure.

### 1.2 电梯演讲（30 秒）

我们不是"给教育产品加了 AI"，而是**用 Agent 重构了教育的底层基础设施**。

UniMind 部署了两个自治 AI Agent：**小宇**是学生的专属学习教练，主动分析薄弱点、规划学习路径、讲解知识难点；**命题官**是教师的对话式教研助手，背后四个 Agent 对抗博弈，一句话完成出题、精修与组卷。

底层是自研的 Memorix 记忆调度算法，用 Weibull 遗忘模型 + 在线梯度下降，让每个学生的记忆模型都在实时进化。

对话即学习，Agent 即基建——这是**教育的基础设施**。

### 1.3 品牌故事（长版）

传统教育 SaaS 是"功能拼盘"——刷题一个模块、课程一个模块、AI 助手一个聊天框。它们各自为政，学生在割裂的体验中疲于切换。

UniMind 的起点不同。我们从第一天就在构建**基础设施层**：

- **Agent 统一运行时**：所有 AI 能力共享一套 Bot → ToolExecutor → ChatService 架构。新增一个 Agent，只需定义工具集和系统提示词——不需要重建流水线。
- **自进化记忆调度**：Memorix 不是静态的间隔重复算法。它用 Weibull 遗忘模型拟合真实遗忘曲线，每个用户的答题行为都实时更新个人记忆参数。算法在你做题的过程中，同时在学习你。
- **对抗式内容生产**：出题不是"AI 生成 → 人工审核"的单次调用。而是 Author、Reviewer、AuthorRevise、Classifier 四个 Agent 迭代博弈，Reviewer 开启深度思考模式，三维质量评分，不达标不放行。

当我们构建了"小宇"学习规划 Agent 和出题工作台 Agent 之后，系统的性质发生了质变——它不再是功能的集合，而是**Agent 的组织**。每个 Agent 有自己的工具链、跨会话记忆、自主决策能力，但共享同一套运行时基础设施。

这就是"新一代智能教育基础设施"的含义：不是给教育加 AI，而是用 AI 重构教育。

## 二、三大 Agent 文案

### 2.1 小宇 — 学习规划 Agent

**一句话**：你的 AI 学习教练，不等你问就主动分析、规划、督促。

**完整描述**：
小宇不是一个问答机器人，而是一个**教练型 Agent**。它能自主调用 17+ 工具：查看你的学习统计、分析知识掌握图谱、检索到期复习、搜索课程和文章，然后综合一切信息给出行动建议。

**自进化能力：**
- **LLM 驱动的自适应指令**：通过分析你的记忆，自动识别学习风格（公式导向/视觉型/例子驱动/记忆技巧型）、认知状态（专注/焦虑/过载/积极），动态调整教学策略
- **Memorix 联动**：基于记忆难度、稳定性、遗忘次数提供精准复习建议，告诉你"为什么总是记不住"以及"该怎么复习"
- **Trajectory 数据收集**：记录每次对话轨迹，为 GEPA（Genetic-Pareto）自进化优化做数据储备

- "你这周微积分正确率下降了，建议先复习洛必达法则的 3 个考点，再做一组中等难度的极限题"
- "你离考研还有 87 天，根据你的薄弱点分布，这是本周的优先级清单"
- "从你的学习数据看，极限这个知识点平均难度 6.5，记忆稳定性只有 3.2 天，建议增加复习频率"

小宇有跨会话记忆：它记得你的目标院校、偏好的学习风格、上次说过的困惑。每次对话，它都在变得更懂你。

**场景文案**：
- 首页引导：「小宇已经分析了你的诊断结果，点击开始第一次规划对话」
- 空状态：「你好，我是小宇。让我看看你的学习数据，帮你理清接下来该做什么。」
- 侧边栏：「小宇 · 学习教练」

### 2.2 出题助手 — 教研 Agent

**一句话**：说一句话，出一套题。背后是四个 AI 的对抗博弈。

**完整描述**：
出题助手是教师的教研 Agent。用自然语言描述需求——"出 10 道微积分极限的客观题，难度 normal"——Agent 自主搜索知识点、调用出题管线、展示结果。说"入库"就存入题库，说"精修"就启动 ARC 四 Agent 对抗管线。

ARC 管线不是简单的"AI 生成 → 过滤"：
- **Author** 生成候选题
- **Reviewer** 开启深度思考模式，从区分度、清晰度、知识覆盖度三维评分
- **AuthorRevise** 根据反馈逐条修改
- **Classifier** 做最终审计：答案正确性、认知层级、难度合规、知识标签

不达标？退回重来，最多 3 轮。最终产出的题目，质量不亚于资深教研团队的产出。

**场景文案**：
- 空状态：「描述你的出题需求，我来处理剩下的」
- 快捷指令：「针对薄弱点出题」「出一套模拟卷」「自定义出题」
- 工作台标题：「出题工作台 · AI 教研引擎」

### 2.3 AI 助教 — 辅导 Agent

**一句话**：真正懂学生的 AI 导师，记得你的每一次对话。

**完整描述**：
AI 助教是面向学生的基础辅导 Agent。它能搜索知识树、查看错题记录、分析薄弱点，更重要的是——它有**跨会话记忆**。它记得你说过"我总是搞混货币乘数和存款乘数"，下次对话时会主动关联。

管理员可以部署多个不同性格和学术背景的 Bot，学生自由切换。每个 Bot 的系统提示词、工具集、性格都在数据库中配置，不需要改代码。

**场景文案**：
- 对话开场：「有什么想问的？我可以帮你查知识点、分析错题、讲解概念。」
- 侧边栏：「AI 助教 · 你的专属导师」

## 三、核心能力文案

### 3.1 Memorix 自进化记忆调度

**标题**：不是间隔重复，是记忆进化。

**描述**：
市面上所有刷题产品都在用 Anki 的 SM-2 或 FSRS 的静态参数。Memorix 不同——它用 Weibull 遗忘模型替代幂律近似，用在线随机梯度下降替代离线批量训练。你每做一道题，你的个人记忆参数就更新一次。算法本身也在学习你。

- 比 FSRS v4.5 预测 RMSE 降低 13.7%
- 用户留存率提升 9.2%
- 真正的"千人千面"——不是推荐不同的题，是用不同的记忆模型调度同一个人的复习

**短文案**：
- 「你的记忆曲线，和别人不一样。Memorix 知道。」
- 「每做一道题，算法也在学习你。」

### 3.2 4-Agent ARC 对抗出题管线

**标题**：四个 AI 博弈，一套高质量考题。

**描述**：
单次 LLM 出题通过率约 60%。ARC 对抗管线通过率 85%+。差距不在模型能力，在**架构**。

Author 写题 → Reviewer 用最强模型 + 深度思考模式逐题审查 → 不达标退回 AuthorRevise 修改 → Classifier 做最终审计（答案正确性、认知层级、难度合规、知识标签）。最多 3 轮迭代，不达标不放行。

**短文案**：
- 「单次 AI 出题通过率 60%。对抗管线 85%+。差距在架构。」
- 「不是 AI 帮你出题，是四个 AI 博弈出题。」

### 3.3 Agent 统一运行时

**标题**：新增一个 Agent，只需写工具和提示词。

**描述**：
UniMind 的所有 AI Agent 共享一套运行时：Bot 模型定义角色 → ToolExecutor 提供工具 → AIChatView 处理对话 → call_ai_with_tools 驱动最多 5 轮自主工具调用。

新增一个 Agent（比如"面试官"或"作业批改员"），只需要：
1. 在 Bot 表创建一条记录
2. 写一个 ToolExecutor 子类
3. 定义系统提示词

不需要重建对话系统、不需要新建 API 端点、不需要改前端。这就是基础设施的意义。

**短文案**：
- 「我们在造操作系统，不是一个 App。」
- 「Agent 即服务。定义工具和提示词，即刻上线。」

### 3.4 三层个性化

**标题**：从冷启动到深度了解，四层递进的个性化。

**描述**：

| 层级 | 机制 | 作用 |
|------|------|------|
| L1 诊断测试 | 首次登录 10 道题快速评估 | 冷启动：初始化知识掌握水平 |
| L2 Memorix | 每次答题实时更新记忆参数 | 行为建模：精准预测遗忘临界点 |
| L3 Agent Memory | 对话中自动提取偏好和习惯 | 深度个性化：越用越懂你 |
| L4 自适应指令 | LLM 分析记忆生成教学策略 | 动态调整：根据认知状态和学习风格优化教学 |

四层叠加，每个学生都有一个独一无二的学习模型——不只是"推荐不同的题"，是"用不同的方式教你"。

**新增 L4 自适应指令详情：**
- **学习风格识别**：公式导向型、视觉型、例子驱动型、记忆技巧型
- **认知状态感知**：专注、焦虑、过载、积极主动
- **动态策略调整**：焦虑学生先共情再给小步骤，过载学生精简内容只聚焦最紧急问题
