# UniMind — 教育机构 AI 基础设施

> 让每位学生拥有个性化学习曲线 | [unimind-ai.com](https://unimind-ai.com)

---

## 核心价值主张

**将「学生遗忘」转化为「系统预判」——在遗忘发生前实施精准干预。**

---

## 机构面临的挑战

学生在培训周期内遗忘率高，复习效率依赖教师个人经验。间隔重复的理论被广泛认可，但 50 名学生 × 1000 道题的手动管理不可实现。结果：提分不稳定 → 续费率波动 → 机构增长受限。

---

## UniMind 解决方案

一套 AI 驱动的教学基础设施，由两个 Agent 和一套算法引擎组成：

**小宇**（学生学习教练）：21 项工具能力。根据每位学生的个性化记忆模型，主动推送到期复习题、批改讲解、渲染知识图谱。学生打开即进入对话，无需学习菜单操作。

**工作台**（教师教研助手）：19 项工具能力。教师通过自然语言出题、管理作业、分析学情。四 Agent 互审管线（Author → Reviewer → Revise → Classifier）确保出题质量。

**Memorix 记忆调度**：基于 Weibull 遗忘模型的在线参数更新。不为学生预设复习计划——系统根据真实遗忘速度动态计算每道题的最佳复习时机。

---

## 差异化

| 传统方案 | UniMind |
|---------|---------|
| 题库 + LMS 功能堆叠 | AI Agent 驱动，对话即操作 |
| 固定间隔复习（1-3-7-30 天） | Memorix 自进化调度，每生每题独立计算 |
| 只看对/错二分 | 错因分类 + IRT 能力值 + 知识点掌握度 |
| 所有机构共享参数 | 机构级数据与模型物理隔离 |
| 学生被动等待任务 | 主动推送到期复习提醒 |

---

## 关键指标

- 预置 **12 学科**知识体系，**11,908 知识点**，四级树状层级
- 学生端 **21 工具**，教师端 **19 工具**，覆盖教学全流程
- **3 种错因**自动分类：概念错误 / 计算失误 / 审题失误
- **机构级隔离**：数据、IRT 参数、知识树、记忆模型完全独立
- **主动推送**：到期题目超过阈值自动通知，支持机构自定义

---

## 目标客户

- **考研/考证培训**：金融 431、CFA、CPA、法考、教资、教育学 311、计算机 408
- **K12 辅导**：高中数学、高中物理
- **医学考试**：USMLE
- **自定义学科**：机构可自建知识树并导入教材

---

## 部署路径

1. 机构注册 → 选择学科模板 → 导入学生
2. 教师出题或导入现有题库
3. 小宇接管复习调度与学习数据分析
4. 教师在维护中心查看班级数据、配置通知策略

---

## 技术概要

Django 6.0 + React 19 + DeepSeek V4 + PostgreSQL + Redis。支持单机与分布式部署。WebSocket 实时通信 + SSE 流式响应。本地 Embedding 模型部署，无需 GPU。
