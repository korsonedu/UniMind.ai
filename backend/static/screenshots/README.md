## 截图技巧
- 使用浏览器全屏模式（F11）让内容区域更大
- Mac：`Cmd+Shift+4` 然后按 Space，点击浏览器窗口，截取无阴影窗口图
- 截完后把 PNG 文件放到这个目录，文件名必须和上面一致
- 刷新 `localhost:5173/intro` 即可在官网看到真实截图
