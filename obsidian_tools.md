# UniMind Agent 工具参考

> 技术附录 | 小宇 21 工具 + 工作台 19 工具 | 版本 2.0

本文档是 UniMind 产品介绍（`PRODUCT.md`）与用户手册（`USER_MANUAL.md`）的技术附录，完整列出两个 AI Agent 的全部工具能力、触发条件及所属意图管线。

---

## 一、小宇 — 学生学习教练（21 工具）

### 1.1 知识检索

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 搜索知识树 | 按关键词检索知识树节点，返回 ID、名称、层级 | "极限是什么意思""导数考点" |
| 题目详情 | 查询单题完整信息（题干、答案、解析） | "这道题为什么选 B" |

### 1.2 学习数据

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 学习统计 | 总做题数、正确率、连续学习天数 | "我学得怎么样" |
| 掌握热力图 | 各知识点掌握率热力数据 | "哪些知识点最强" |
| 薄弱知识点 | 错误率最高知识点排名 | "我哪里学得最差" |
| 错题列表 | 最近作答错误题目 | "最近错了哪些题" |
| 考试历史 | 历次测验与考试成绩 | "上次考试多少分" |
| 难度分析 | 知识点遗忘曲线与记忆稳定性 | "为什么总是记不住" |

### 1.3 练习与测试

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 抽题练习 | 按知识点/难度从题库抽题 | "出 5 道极限题" |
| 到期复习 | 今日到期间隔重复复习任务 | "今天要复习什么" |
| 诊断测试 | 新用户水平评估与记忆模型初始化 | （首次登录自动触发） |

### 1.4 学习计划

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 当前计划 | 查看活跃计划标题与进度 | "计划怎么样了" |
| 创建计划 | 新建计划，自动归档旧计划 | "制定高数复习计划" |
| 更新任务 | 标记计划中单任务状态 | "极限做完了" |

### 1.5 资源与批改

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 搜索课程 | 检索视频课程 | "有没有微积分的课" |
| 搜索字幕 | 定位课程视频中知识点讲解时间点 | "哪个视频讲了洛必达" |
| 搜索文章 | 检索深度文章 | "推荐学习方法文章" |
| 批改答案 | 主观题批改与解析反馈 | "帮我批这道题" |

### 1.6 班级数据

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 班级薄弱分析 | 全班知识点正确率分布 | "我们班哪弱" |
| 班级表现摘要 | 全班均分与活跃度 | "班级整体怎么样" |

### 1.7 可视化

| 工具 | 功能 | 说明 |
|------|------|------|
| 渲染可视化 | 在对话流中渲染数据卡片与图表 | Agent 自主调用 |

## 二、工作台 — 教师教研助手（19 工具）

### 2.1 出题

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 搜索知识点 | 搜索知识点或知识树结构 | "搜索极限相关知识点" |
| 快速出题 | AI 单步生成新题，5-10 秒 | "出 5 道极限题" |
| 题目入库 | 将生成题目正式存入题库 | "入库""存下来" |
| ARC 精修 | 启动四 Agent 互审管线 | "精修""ARC 审核" |
| 管线状态 | 查询管线执行进度 | "跑完没""进度" |

### 2.2 题库管理

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 浏览题目 | 搜索或随机抽取已有题目 | "看看题库""随机抽 5 道集合题" |
| 题库统计 | 总题数、题型分布、出题偏好 | "题库统计""出了多少题" |

### 2.3 班级管理

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 班级列表 | 列出机构下所有班级 | "有哪些班""查火箭班" |
| 班级薄弱分析 | 全班知识点正确率与 Top 5 薄弱点 | "班级薄弱点""学情分析" |
| 班级成绩册 | 学生 × 作业得分矩阵 | "火箭班成绩册" |

### 2.4 作业

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 布置作业 | 创建作业并发布至指定班级 | "布置给火箭班" |
| 作业进度 | 作业提交人数与批改状态 | "作业交了没" |
| 批改作业 | 为单个提交打分 | "给 XX 打 85 分" |

### 2.5 学生管理

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 学生详情 | 单学生正确率、薄弱点、做题记录 | "张三学得怎么样" |
| 发送通知 | 向指定学生发送站内提醒 | "提醒张三交作业" |

### 2.6 资产

| 工具 | 功能 | 触发示例 |
|------|------|---------|
| 浏览课程 | 检索机构课程库 | "有什么课" |
| 浏览文章 | 检索机构文章库 | "看看文章" |
| 分配课程 | 将课程分配给指定班级 | "给火箭班加微积分课" |

### 2.7 可视化

| 工具 | 功能 | 说明 |
|------|------|------|
| 渲染卡片 | 在对话中展示交互式确认与数据卡片 | Agent 自主调用 |

## 三、意图管线

意图管线通过 Embedding 检索 + 关键词匹配对工具进行预筛选，将模型可调用工具从全集缩减至场景相关子集，降低路由错误率。

### 3.1 小宇管线（7 条）

| 意图 | 关键词 | 可用工具 |
|------|--------|---------|
| 学习规划 | 规划、安排、计划、复习、备考 | 8 |
| 数据分析 | 分析、掌握率、薄弱、趋势、成绩 | 9 |
| 出题练习 | 出题、做题、练习、测试、刷题 | 7 |
| 学习资源 | 课程、视频、文章、推荐 | 4 |
| 错题回顾 | 错题、做错了、哪里错、错因 | 4 |
| 面板数据 | 面板、仪表盘 | 1 |
| 批改评分 | 批改、判分、对不对、帮我改 | 3 |

### 3.2 工作台管线（8 条）

| 意图 | 关键词 | 可用工具 |
|------|--------|---------|
| 薄弱分析 | 薄弱、弱项、学情、正确率低 | 4 |
| 精修 | 精修、ARC、润色、对抗审核 | 3 |
| 出题生成 | 出题、生成、新题、再来、调整 | 7 |
| 管线状态 | 进度、跑完没、状态 | 1 |
| 题库统计 | 统计、多少题、分布 | 1 |
| 学生查询 | 学生、学得怎么样、看一下 | 4 |
| 布置作业 | 作业、布置、发布、下发、选题 | 7 |
| 浏览资产 | 看看课程、有什么题、浏览 | 4 |
