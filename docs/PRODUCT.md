# UniMind — 教育机构 AI 基础设施

UniMind 是一套面向教育培训机构的智能教学系统。它在题库和 LMS 之上构建了一层 AI 调度引擎，核心能力是 Memorix 记忆调度算法：基于每个学生每道题的实时数据，动态计算最佳复习时机，在遗忘临界点精准切入。

---

## 产品结构

**学生端 — 小宇**：AI 学习教练，拥有 17 个工具。自动抽题、评分、讲解、渲染知识图谱。学生登录后小宇主动推送到期复习题，每道题独立计算间隔，错因自动分类（概念错误、计算失误、粗心），不同错误类型采用不同复习策略。

**教师端 — 工作台**：命题官 Agent 负责出题。教师指定知识点、题型和难度，一句话生成题目。四 Agent 互审管线（Author → Reviewer → Revise → Classifier）将单次出题通过率从 60% 提升至 85% 以上。工作台同时提供班级数据分析、薄弱点分布、IRT 能力值对比。

---

## 算法引擎

**Memorix 记忆调度**：基于 Weibull 遗忘模型的在线 SGD 参数更新。图扩散（Memorix-Field）将知识图谱结构融入调度，跨知识点传播难度。再巩固窗口（REC）在记忆不稳定期精确切入复习，替代固定间隔。

**IRT 认知诊断**：三参数 Logistic 模型，机构级参数隔离。同一道题在不同机构有独立的区分度、难度和猜测参数。MML-EM 参数估计加 θ 能力值输出，按知识点粒度提供掌握度分解。

**变式题匹配**：学生做错后自动检索同知识点相近难度的变式题，插入下次复习队列。

**GEPA 自进化**：Trajectory 数据收集后周度分析工具调用成功率和对话完成度，自动生成优化建议存储至 Redis 等待人工审核。

---

## 知识图谱

预置 12 个学科共 11,908 个知识点，四级树状结构（学科 → 篇章 → 小节 → 考点）。机构可导入自定义知识树，题目必须关联知识点。

---

## 机构能力

机构间数据物理隔离，包括学生记忆模型、IRT 参数、知识图谱定制。支持白标（自定义 Logo 和域名）和 API 接入。到期复习通知推送支持机构自定义开关、渠道和触发阈值。
