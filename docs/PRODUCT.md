# UniMind 产品白皮书

凌晨一点，一个考研机构的老板在群里催老师交下周的模拟卷。另一个老师私信他，说上周的错题统计还没做出来。他关掉手机，想起上个月流失的三个学生，家长的理由几乎一样：「孩子做了很多题，分数没变化。」

这不是个案。

我叫清言，在 UniMind 负责产品。过去大半年我跑了几十家教培机构，听到最多的一句话是「我知道每个学生不一样，但我没有工具去做到不一样」。机构买过题库、搭过 LMS、试过各种 AI 答题机器人，最后学生还是在同一个池子里刷同一批题，老师还是凭感觉判断谁哪里弱。

UniMind 做的是题库和 LMS 之上的一层智能调度系统。题库存题目，LMS 管流程，UniMind 决定什么时间、给哪个学生、推哪道题。这个决定跑在算法引擎上，靠每个学生的实时记忆参数驱动，不是靠定时脚本赶出来的。

学生端叫小宇，一个 AI 学习教练，手里拿着 17 个工具。它能查你的遗忘曲线、调取你的错题本、分析你的错因类型，概念没懂、算错了、还是粗心，然后从知识图谱里找出关联薄弱点，给你推一道刚刚好的题。学生打开 App，小宇已经在说话了：「三角函数你有 3 道到期题，上次导数正确率从 40% 提到 65% 了，今天趁热继续。」学生每做一道题，小宇就更新一次这个学生的记忆参数，遗忘速度是每个人每道题独立计算的，跟隔壁桌的同学没关系。

教师端是工作台。最核心的模块叫命题官，你跟它说「出 10 道计量经济学选择题，难度中上，侧重异方差和自相关」，它就启动一套四个 Agent 互相挑错的管线。第一个 Agent 出题，第二个 Agent 用深度思考模式打三维质量分，不满意就打回去让第一个改，第三个 Agent 做最终审计，三道关卡过了题才入库。这一套下来，单次 LLM 出题的及格率从 60% 拉到 85% 以上。教师还能在工作台里看到班级 IRT 能力值分布、高频错题榜、知识点掌握率热力图。机构可以自定义知识图谱，我们预置了 12 个学科共 11,908 个知识点，你也可以上传自己的教材建自己的树。

底层算法做三件事。第一，Memorix 记忆调度基于遗忘模型在线更新参数，还融合了知识图谱的结构信息，一个知识点被强化，相邻知识点也受益。再巩固窗口机制在记忆最不稳定的时间点精准插入复习。第二，IRT 认知诊断是机构级隔离的，同一道题在重点中学和普通中学有独立的区分度参数，不会互相污染。第三，GEPA 自进化管线每周自动分析工具调用成功率和对话完成度，生成优化建议推到 Redis 等人工审核，不会自己偷偷改任何东西。

数据隔离是机构客户最关心的事。每家机构的数据、知识图谱定制、IRT 参数、学生记忆模型全部物理隔离。你要白标，把 Logo 和域名换成你自己的。你要 API，可以把出题和调度能力接进你现有的系统。

价格三档。Free 方案支持 30 个学生，够一个小体量考研辅导班先用起来。Growth 方案 200 个学生。Pro 方案不限学生数。所有方案都带机构隔离和完整的算法能力，不搞功能阉割。

到期复习的通知推送，机构侧可以自己配置开关、渠道和阈值。一个考研机构设成每天上午八点飞书推送，另一个 K12 机构设成到期题超过两道就微信提醒。这些不用找我改，你自己在工作台配。

回到开头那个凌晨还在催卷子的老板。他缺的不是题目，是题目和学生的匹配没有颗粒度。UniMind 做的事就是把颗粒度做到每个人每道题。一个机构如果能让学生感觉到「这个东西真的知道我不会什么」，续费率就不是靠销售技巧撑着的。我看到的那些续费率超过 80% 的机构，没一个是靠话术做到的。
