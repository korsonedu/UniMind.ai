# 事故记录：QA 列表接口 500

- 日期：2026-04-30
- 影响：`GET /api/qa/questions/` 返回 500，答疑页不可用。

## 1. 复现
- 认证用户调用：`GET /api/qa/questions/?filter=all&search=`
- 结果：500

## 2. 根因证据
- 异常：`django.db.utils.OperationalError: no such column: faq_system_answer.updated_at`
- 触发点：`faq_system.views.QuestionListCreateView` 列表查询 + 预取 `answers` 后序列化。

## 3. 修复
- 新增迁移：`backend/faq_system/migrations/0001_initial.py`
- 新增迁移：`backend/faq_system/migrations/0002_answer_updated_at.py`
- 迁移操作：补 `faq_system_answer.updated_at`，并回填历史数据（`updated_at = created_at`）。

## 4. 验证
- `GET /api/qa/questions/?filter=all&search=` -> 200
- `GET /api/qa/questions/?filter=unsolved&search=` -> 200
- `POST /api/qa/questions/` -> 201

## 5. 防再发
- 模型变更必须附迁移文件。
- CI 增加迁移漂移检查：`python3 manage.py makemigrations --check --dry-run`。
