## P3 | UniMind 是什么

**一句话：** 输入考点，AI 出题；学生扫码，即刻练习；系统自动复习、批改、出报告。

**核心能力（6 个图标卡片）：**
1. AI 出题 — 10 秒生成一道高质量题目，85% 可直接用于课堂
2. 自适应复习 — 基于遗忘曲线的个性化复习调度
3. AI 批改 — 主观题自动评分 + 解析
4. 知识图谱 — 可视化学生薄弱环节
5. AI 助教 — 7x24 在线答疑
6. 学情报告 — 每周自动生成，家长可看

**底部：** 你负责教课，UniMind 负责其他所有事。

## P4 | 核心功能 ①：AI 对抗出题引擎

**标题：** 不是"AI 随便出题"，是 4 个 AI Agent 互相审核

**流程图（简化版）：**
```
考点输入 → [Author 出题] → [Reviewer 审核] → [修正] → [Classifier 分类] → 入库
                                ↑        ↓
                          不达标 ← 最多 3 轮 → 达标
```

**关键数据：**
- 单次 LLM 出题可用率：~60%
- UniMind ARC 管线可用率：**85%+**
- 支持题型：单选/多选/判断/名词解释/简答/论述/计算
- 支持难度控制：入门/简单/一般/困难/地狱，由教师锁定，AI 不能擅自更改

## P5 | 核心功能 ②：Memorix 自适应复习

**标题：** 不是"艾宾浩斯曲线"，是为每个学生实时学习的算法

**对比表：**

| | 传统间隔重复 | FSRS v4.5（开源最优） | **Memorix（UniMind）** |
|---|---|---|---|
| 遗忘模型 | 固定曲线 | 幂律分布 | Weibull 分布（可学习形状参数） |
| 个性化 | 无 | 全局参数 | 每个学生独立 20 维权重向量 |
| 更新方式 | 无 | 批量拟合 | 每次答题后实时在线学习 |
| 预测精度 | 差 | RMSE 基线 | **比 FSRS 低 13.7%** |
| 用户留存 | 基线 | 基线 | **提升 9.2%** |

**一句话：** 学生每做一道题，系统就更了解他一次。复习越用越准。

**数据来源：** 431 金融考研数据集，500+ 用户，120,000+ 条复习记录。

## P6 | 核心功能 ③：知识图谱 + 学情分析

**标题：** 红黄绿一眼看清，谁在哪个知识点上卡住了

**功能点：**
- 4 层知识树：学科 → 章 → 节 → 知识点
- 每个知识点：掌握度评分 + 热力图（绿/黄/红）
- 自动识别瓶颈：DAG 遍历找到前置知识缺口
- 支持用户自标注：优先级、置信度、标签、笔记

**场景举例：**
> 学生在"投资组合管理"反复出错 → 系统发现前置知识"资产定价模型"掌握度仅 35% → 自动推荐回到 CAPM 复习
