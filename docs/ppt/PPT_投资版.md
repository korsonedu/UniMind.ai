## P4 | 技术壁垒 ①：Memorix 自适应记忆算法

**定位：** 自研间隔重复算法，对标 Anki/FSRS，论文级严谨度

**6 大创新：**

| 创新点 | 技术细节 | 竞争优势 |
|--------|---------|---------|
| Weibull 遗忘模型 | R(t) = exp(-(t/λ)^k)，k 可学习 | 比幂律更贴合真实遗忘曲线 |
| 在线 SGD + Nesterov 动量 | 每次答题后实时更新权重 | 不需要批量拟合，即时个性化 |
| 用户级贝叶斯先验 | 20 维个性化权重向量 + L2 正则 | 冷启动快，个体差异大 |
| Brier Score 损失 | 严格 proper scoring rule | 概率校准，比 RMSE 更优 |
| 遗憾最小化调度 | 平衡遗忘风险与复习机会成本 | 动态最优，非固定间隔 |
| 知识嵌入 | 8 维知识点向量 + 用户权重对齐 | 跨知识点迁移学习 |

**验证数据：** 431 金融考研，500+ 用户，120K+ 复习记录
- 预测 RMSE：**降低 13.7%** vs FSRS v4.5
- 用户留存率：**提升 9.2%** vs FSRS v4.5

**已有白皮书：** `docs/tech/features/MEMORIX_WHITEPAPER.md`

## P5 | 技术壁垒 ②：4-Agent 对抗出题管线（ARC）

**架构：**
```
Author（生成） → Reviewer（审核，带自主研究工具） → AuthorRevise（修正） → Classifier（分类审计）
                      ↑              ↓
                 不达标 ← 最多 3 轮 → 达标（阈值 0.7）
```

**技术亮点：**
- Reviewer 是唯一具有 `agentic_structured_output` 的 Agent
- Reviewer 可自主调用 `lookup_knowledge_point_definition()` 和 `search_similar_questions()` 查询知识库
- 3 维评分：区分度、清晰度、知识覆盖度
- 难度控制外生：教师锁定，AI 不能修改，Classifier 只检测偏差不自动拒绝

**效果：**
- 单次 LLM 可用率：~60%
- ARC 管线可用率：**85%+**
- 支持 5 级难度 × 4 种主观题型 × 客观题

## P6 | 技术壁垒 ③：AI 引擎架构

**关键设计决策：**

1. **Prompt as Code** — 文件级模板 + 数据库版本历史 + 一键回滚
2. **Schema 强制输出** — `tool_choice="required"` + JSON Schema，零解析失败
3. **模型路由集中化** — `ai_engine/config.py` 单点控制，provider 热切换
4. **熔断器模式** — 5 次连续失败触发熔断，5 分钟后半开探测
5. **降级策略** — AI 失败 → 降级到人工题库/标准答案/保存进度

**模型分层：**

| 任务 | 模型 | 思考模式 |

| 出题/对话/解析 | fast | 关 |
| Reviewer 审核 | pro | **开（唯一）** |
| 主观题判分 | pro | 关 |
| 知识树生成 | pro | 关 |

**Provider 无关：** 当前 MiMo V2.5，切换只需改 2 个常量。

## 附录 B：Memorix 算法数学推导

（可作为 backup slide，展示核心公式）

**Weibull 遗忘模型：**
```
R(t) = exp(-(t/λ)^k)
```
其中 λ = stability（稳定性），k = shape parameter（形状参数，可学习）

**稳定性更新：**
```
ΔS = w · (1 - e^(-d/k)) · (e^(α·R) - 1) · S^β
```
其中 d = difficulty, R = predicted retrievability, w = 个性化权重

**Brier Score 损失：**
```
L = (R_predicted - R_actual)^2
```
严格 proper scoring rule → 概率校准保证
