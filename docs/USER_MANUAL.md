## 一、系统概述

UniMind 由两个 AI Agent 驱动：

- **小宇**（学生端）：AI 学习教练，负责练习推送、批改讲解、复习调度、学习数据分析
- **工作台**（教师端）：AI 教研助手，负责出题、作业管理、学情分析、班级运营

学生与小宇对话完成全部学习活动；教师在工作台对话中完成教学管理。系统后端自动运行 Memorix 记忆调度引擎，为每位学生每道题计算个性化复习时机。

## 二、快速入门

### 2.1 机构注册与配置

1. 注册机构账号，选择学科模板或导入自定义知识树
2. 创建班级，邀请教师加入
3. 教师导入或生成题库
4. 邀请学生加入班级

### 2.2 学生首次登录

1. 使用机构提供的邀请链接注册
2. 完成诊断测试（系统自动评估当前水平）
3. 进入小宇对话页，开始学习

### 2.3 教师首次登录

1. 使用机构管理员分配的账号登录
2. 进入工作台对话页
3. 根据引导完成题库初始化（可选）
4. 开始出题与管理

## 三、学生端：小宇 AI 学习教练

### 3.1 对话交互

学生与小宇的交互通过自然语言对话完成。小宇根据学生的记忆模型、薄弱点和学习历史，主动提供以下服务。

### 3.2 练习与复习

**开始练习**
向小宇表达练习需求：
- "给我出几道极限的题"
- "我想做微积分的练习"
- "今天有什么要复习的"

**查看复习任务**
- "有哪些到期要复习的题"
- "今天要复习什么"

**错题回顾**
- "我最近错了哪些题"
- "看看我的错题"
- "这道题为什么错了"

小宇会展示每道错题的题干、正确答案、解析以及错误类型分类（概念错误 / 计算失误 / 审题失误）。

### 3.3 学习数据分析

向小宇查询学习数据：

| 查询意图 | 示例对话 |
|---------|---------|
| 整体概览 | "我学得怎么样""学习报告" |
| 知识点掌握 | "哪些知识点我掌握得好""薄弱环节" |
| 正确率趋势 | "最近正确率有提升吗""成绩趋势" |
| 考试历史 | "上次考试多少分" |
| 遗忘分析 | "为什么我总是记不住""我需要加强哪些" |

小宇以对话文本和可视化卡片形式呈现分析结果。

### 3.4 学习计划

**创建计划**
- "帮我制定一个高数复习计划，两周完成"
- "安排一下考研数学的复习"

**查看进度**
- "我的计划进展怎么样了"
- "还有多少没完成"

**更新任务**
- "极限那章我做完了"
- "跳过今天的导数练习"

小宇自动归档旧计划，每次只保留一个活跃计划。

### 3.5 资源检索

| 资源类型 | 示例对话 |
|---------|---------|
| 视频课程 | "有没有讲微积分的课" |
| 视频定位 | "哪个视频讲了洛必达法则" |
| 文章推荐 | "推荐一些学习方法文章" |

## 四、教师端：工作台

### 4.1 对话交互

教师在工作台对话页通过自然语言完成所有教学管理操作。工作台 Agent 理解教学语境，自动调用相应工具。

### 4.2 出题

**快速出题**
1. 先搜索知识点确认知识范围："搜索极限相关知识点"
2. 指定出题需求："出 5 道极限的计算题，中等难度"
3. 题目生成后在左侧面板查看，可直接编辑或删除
4. 确认后入库："存下来"或点击面板「存入题库」按钮

**调整题目**
- "换一道""再来一组""出几道难的"
- "把第 3 题改简单一点"

**ARC 精修出题**
1. "精修这几道题"或"ARC 对抗审核"
2. 系统启动四 Agent 互审管线，逐题审核优化
3. "进度怎么样"查询管线状态
4. 完成后查看精修结果

### 4.3 题库管理

**浏览题目**
- "看看题库""有没有集合的题"
- "查看极限相关的所有题目"

**随机抽题**
- "随机抽 5 道集合的题"
- "从题库抽 10 道极限题"

**题库统计**
- "题库统计""出了多少题""出题分布"

### 4.4 作业管理

**布置作业**
1. 先获取题目（从题库抽取或新生成）：
   - 抽题："随机抽 5 道集合题"
   - 出题："出 3 道极限计算题"→ 入库
2. 指定班级与标题：
   - "布置给火箭班，标题'集合练习'"
3. 系统展示确认卡片，点击确认发布

**查看进度**
- "作业交了没""集合练习进度"
- "还有谁没交作业"

**批改作业**
- "批改张三提交的作业"
- "批改作业 #3"

### 4.5 学情分析

**查看学生**
- "张三学得怎么样"
- "看一下李四的学习数据"

返回：正确率、薄弱知识点、近期做题记录、作业提交情况。

**班级分析**
- "班级薄弱点在哪""学情分析"
- "看看火箭班的成绩""成绩册"

返回：全班知识点正确率分布、最薄弱知识点 Top 5、学生 × 作业得分矩阵。

### 4.6 班级运营

**管理班级**
- "有哪些班""查一下火箭班""班级列表"

**分配课程**
- "给火箭班加上微积分课"
- "把视频课 123 分配给火箭班"

**发送提醒**
- "提醒一下张三交作业"
- "通知火箭班明天有测验"

## 五、机构管理

### 5.1 班级管理

在维护中心创建、编辑、删除班级，管理班级成员。支持按班级查看学习数据与分析报告。

### 5.2 教师管理

邀请教师加入机构，分配角色权限。支持多教师协作与数据共享。

### 5.3 知识树管理

机构可导入自定义知识树（Markdown 格式），覆盖或补充系统预置学科。知识树支持版本管理。

### 5.4 通知配置

配置复习提醒的推送开关、推送渠道和触发阈值。支持按班级粒度差异化设置。

### 5.5 品牌白标（Pro 计划）

自定义机构 Logo、域名和配色方案，实现品牌统一。

## 六、常见问题

**Q: 学生看不到复习提醒？**
A: 确认机构管理员已在维护中心开启通知推送，并检查触发阈值设置。

**Q: 出题质量不满足要求？**
A: 使用 ARC 精修管线对题目进行深度优化。系统会在管线完成后展示修改对比。

**Q: 知识树如何自定义？**
A: 在维护中心上传 Markdown 格式的知识树文件，系统自动解析导入。格式见知识树模板。

**Q: 支持哪些学科？**
A: 预置金融 431、高中数学、高中物理等 12 个学科。机构可自定义学科。

**Q: 数据存储在哪里？**
A: 所有数据存储于机构专属数据库，机构间物理隔离。支持私有化部署需求。
