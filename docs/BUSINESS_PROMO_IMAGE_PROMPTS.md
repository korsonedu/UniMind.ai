## 使用说明

- 以下 prompt 面向 **Midjourney v6 / DALL-E 3 / Stable Diffusion XL** 优化
- 标注 `[MJ]` = Midjourney 专用，`[DE]` = DALL-E 专用，`[通用]` = 均可
- 每条 prompt 后标注**适用场景**和**建议尺寸**
- 生成后建议用 Canva/Figma 叠加中文文案

## 六、Midjourney 通用后缀参数

所有 [MJ] prompt 末尾建议添加：

```
--ar 16:9 --style raw --v 6.0 --s 250 --q 2
```

社交竖版图替换为：

```
--ar 3:4 --style raw --v 6.0 --s 150
```

## 七、批量生成工作流

1. **生成底图**：用上述 prompt 在 Midjourney/DALL-E 生成 4-8 张候选
2. **挑选 + Upscale**：选 2-3 张最佳的进行放大
3. **去 AI 感**：用 Photoshop/Canva 微调色彩、去除 AI 痕迹（手指、文字乱码等）
4. **叠加文案**：在 Canva 中添加中文标题、副标题、CTA 按钮
5. **导出多尺寸**：1920×900（PC）、1080×1440（小红书竖版）、900×500（公众号）
